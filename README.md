## Technical test

### Versions

- node : v18.16.1
- npm : 9.5.1
- nvm : 0.39.1

### Run App

```
npm run start
```

### Run unit test

```
npm run test
```

### see coverage

1. `npm run test`
2. a folder `/coverage` will be created
3. in the route `/coverage/lcov-report/index.html`
4. open the `index.html` with the preferred browser
